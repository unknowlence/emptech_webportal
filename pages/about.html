About This Web Portal

This is a web portal assignment designed solely based on the Lead Developer's imagination. This website is intended to complete the requirements for the subject "Empowering Technology". Though this may be an assigned activity, this is actually somewhat enjoyable to be able to create something from scratch.

Features

The home page includes a search function that directs queries to Google's search engine, alongside some funny inside jokes relating to Honkai Star Rail. The interface is intentionally made to be simple, because the Lead Developer is not that experienced in web development and is just following youtube tutorials.

Navigation

The top section provides quick access to various sections of the website, including the Home, About, and Organization pages. There is also some additional content available through the profile icon.
