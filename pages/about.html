About This Web Portal

This is a web portal assignment designed to replicate the look and feel of Google's search interface. The portal features a clean, minimalist design with essential navigation elements at the top.

Features

The main page includes a search function that directs queries to Google's search engine. Users can perform standard searches or use the "I'm Feeling Lucky" feature to go directly to the first result. The interface maintains a simple and intuitive layout.

Navigation

The top section provides quick access to various Google services including Gmail, Images, and other apps. Links to Home and About pages are available on the left side for easy navigation throughout the portal.
